# Notice

SmartSurf Safety Platform is a SmartSurf-branded customization built on top of Traccar Web and the official Traccar Docker image.

Upstream projects:

- Traccar: https://github.com/traccar/traccar
- Traccar Web: https://github.com/traccar/traccar-web

This repository keeps SmartSurf product code, deployment templates, and documentation public while excluding production-only runtime data and secrets.

Senlay integration is represented through API/proxy calls. Production Senlay credentials must stay server-side and must not be committed.
